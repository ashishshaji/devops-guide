Playbook Architecture

A Playbook is a collection or list of "plays" targeting your inventory.

The Play

Defines activities for specific groups (e.g., "DB Server Play" and "App Server Play").

  • Target Hosts
  • Remote User

Become Privilege

Setting become: true allows the play to execute commands with root/sudo privileges.

Deploying a Web Server

1

Install Apache apt module

- name: Install Apache
  apt:
    name: apache2
    state: present
2

Deploy HTML copy module

Moves index.html from Control Node to /var/www/html.

- name: Copy index page
  copy:
    src: index.html
    dest: /var/www/html/index.html

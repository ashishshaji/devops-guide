Manual vs. Ansible Workflow

Manual Method (Legacy)

  • Individual SSH into every single server.
  • Cryptic one-liner commands without context.
  • Knowledge exists only in the developer's head.

Ansible Method (Modern)

  • Logic written once in an Ansible Playbook.
  • Self-explanatory task names (e.g., "Install Python").
  • The Playbook serves as living documentation.
Agentless Design

Ansible Architecture

Ansible uses a Push Architecture. Unlike Chef or Puppet, you don't need to install any agent software on your remote servers.

Installation Guide

Windows

Uses **WSL (Windows Subsystem for Linux)** to run an Ubuntu environment.

# Inside WSL terminal
sudo apt update

Ubuntu

Direct installation via the apt package manager.

sudo apt update
sudo apt install ansible -y

Mac OS

Installation via the popular Brew package manager.

brew install ansible

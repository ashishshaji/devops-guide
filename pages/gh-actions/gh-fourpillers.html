Real-World Implementation

1 Using Contexts in Workflows

name: CI Environment Check on: [push] jobs: info: runs-on: ubuntu-latest steps: - run: echo "Repo name: ${{ github.repository }}" - run: echo "Current branch: ${{ github.ref_name }}"

2 Conditions & Variables

jobs: deploy: if: ${{ github.ref == 'refs/heads/main' }} runs-on: ubuntu-latest env: TARGET: "production" steps: - run: ./deploy.sh $TARGET
